# Paper

## Paper Tasks
This section is a clone of a check list currently used in Trello. However, the information might be interesting for anyone writing their first paper.

### writing
- Fill out the Trello [Template](https://trello.com/b/KitYzlPS/3s-papers)
- Pick a Target Conference/Workshop/Journal for your paper, now you also have a deadline

### Submission
- Submit paper
- upload preprint to arXiv (in case there is no double-blind review process)

### Camera Ready
- add reviewer comments to your paper repository
- submit camera ready version
- update preprint on arXiv
- clean up code repository and add a README with citation information (preprint version is okay)
- announced acceptance of the paper on website (ask Nils/Tim)

### Conference preparation
- register at conference, travel, hotel
- ask David which expense account to use
- submit "Antrag auf Durchf√ºhrung einer Dienstreise" (Application for Work Trip)
- apply for visa (if required)
- get "A1 Bescheinigung" for travel insurance (EU only)

### Wrap-Up
- notify David about "Abschlagszahlung"
- prepare expense report
- notify David about final cost
- update bibliography (e.g., with DOI) in mcc.bib
